 📋 About
This intelligent PDF ChatBot application leverages 
Retrieval-Augmented Generation (RAG) and a Dual LLM 
Architecture powered by Groq's high-performance API. 
Users can upload any PDF document and engage in natural 
conversations about its content. The system uses PyMuPDF 
for text extraction, LangChain for intelligent text 
chunking, Sentence Transformers for semantic embeddings, 
and FAISS for millisecond-level vector search. Two 
specialized LLMs work in sequence - LLM 1 (llama-3.3-70b) 
summarizes retrieved context while LLM 2 (llama-3.1-8b-instant) 
generates accurate, conversational responses with 
full conversation memory support.
